Respuesta:Un virtual host o vhost es un proveedor de servicios web que incluyen funcionalidad de servidor y servicios de conexion a internet. Es usado cuando los individuos o companias no desean comprar/mantener sus propios Web Servers. De esta forma el vhost le provee al cliente registro de nombres de dominio, almacenamiento de archivos y servicios de direccionamiento. Como el individuo utiliza este tipo de hosting virtual, compite por recursos web con las demas personas que utilizan el mismo proveedor de virtual hosting.
